# Homebrew Tap for Planetary Escape

Homebrew formulae for [Planetary Escape](https://planetaryescape.dev) tools.

## Available Formulae

### blah

Terminal client for [blah.chat](https://blah.chat) AI assistant.

```bash
brew install planetaryescape/tap/blah
```

## Usage

```bash
# Add the tap
brew tap planetaryescape/tap

# Install blah CLI
brew install blah

# Verify installation
blah --version

# Login to your account
blah login
```

## Updating

```bash
brew update
brew upgrade blah
```
